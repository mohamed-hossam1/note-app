Can you make the model fail?

Welcome to the Massive Multi-discipline Multimodal Understanding and Reasoning (MMMU) Mocha Project! In this project, your goal is to create complex image-based prompts that will make the model fail.

Once you've successfully caused the model to fail, you will simply add the correct final answer along with a brief explanation of what was wrong and what you fixed. If you did not make the model fail, you need to try 3 times or until the model fails.

Update 12/18

When the model fails, you will no longer need to edit the models response. Instead, you will simply add the correct final answer along with a brief explanation.

Workflow

SKIP the task if you are unsure about how to ask a domain-specific question about the image / it falls outside of your domain expertise.

  1. Assess the validity of the image (Hint: Most images will be valid)

  2. Submit a valid prompt for the category and difficulty level. Come up with your own prompts or refer to provided examples!

    Pin the image and the prompt while you're reviewing the response to ensure accuracy and attention to detail!

  3. Determine if the model failed/answered incorrectly. If it fails, move to step 4. If it does not fail, try another prompt until it fails (continue trying up to 3 prompts).

  4. Extract the correct final answer

  5. Classify the difficulty level of the last prompt
